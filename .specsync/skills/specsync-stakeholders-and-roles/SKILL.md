# Stakeholders and Roles

Follow the shared workflow in [../_shared/document-maintenance.md](../_shared/document-maintenance.md).

Review and own:

- `specs/01_Project-Overview/Stakeholders-and-Roles.md`

When the orchestrator agent asks for input:

- say whether the proposal affects this file
- list the file you reviewed
- if affected, describe the exact stakeholders delta to stage

Update this document when the request changes:

- stakeholders, roles, teams, or ownership
- decision authority or accountability
- communication cadence or format
- escalation contacts or paths

Do not edit this document for product requirements or technical design unless team ownership itself changes.
