# Requirements Traceability Matrix

Follow the shared workflow in [../_shared/document-maintenance.md](../_shared/document-maintenance.md).

Review and own:

- `specs/02_Product-Specification/Requirements-Traceability-Matrix.md`

When the orchestrator agent asks for input:

- say whether the proposal affects this file
- list the file you reviewed
- if affected, describe the exact traceability delta to stage

Update this document when the request changes:

- requirement identifiers or summaries
- links from requirements to design artifacts
- links from requirements to implementation artifacts
- links from requirements to validation evidence
- status of coverage gaps

Do not restate the full requirement text here when the PRD already owns it. Keep this file relational and traceable.
