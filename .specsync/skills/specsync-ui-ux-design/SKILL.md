# UI UX Design

Follow the shared workflow in [../_shared/document-maintenance.md](../_shared/document-maintenance.md).

Review and own:

- `specs/03_Architecture-and-Design/UI-UX-Design.md`

When the orchestrator agent asks for input:

- say whether the proposal affects this file
- list the file you reviewed
- if affected, describe the exact UI-UX delta to stage

Update this document when the request changes:

- screens, views, or interaction flows
- content and accessibility rules
- links to Figma, prototypes, or other design assets
- review status of UX artifacts

Do not use markdown as the primary mockup medium when a richer design asset is available. This document should point to the source of truth.
