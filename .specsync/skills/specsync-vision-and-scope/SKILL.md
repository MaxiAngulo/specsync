# Vision and Scope

Follow the shared workflow in [../_shared/document-maintenance.md](../_shared/document-maintenance.md).

Review and own:

- `specs/01_Project-Overview/Vision-and-Scope.md`

When the orchestrator agent asks for input:

- say whether the proposal affects this file
- list the file you reviewed
- if affected, describe the exact vision-and-scope delta to stage

Update this document when the request changes:

- the product or initiative vision
- the problem statement
- goals, non-goals, or success metrics
- in-scope or out-of-scope boundaries
- project-level risks, assumptions, or open questions

Do not edit this document for detailed functional behavior, architecture, or test execution details unless they alter project scope or goals.
