# Runbook

Follow the shared workflow in [../_shared/document-maintenance.md](../_shared/document-maintenance.md).

Review and own:

- `specs/05_Operations-and-Delivery/Runbook.md`

When the orchestrator agent asks for input:

- if affected, describe the exact runbook delta to stage

Update this document when the request changes:

- service ownership or operational dependencies
- dashboards, alerts, logs, or support tooling
- startup, shutdown, troubleshooting, or incident procedures
- escalation contacts and operational actions

Do not edit this document for product requirements unless operator behavior or operational readiness changes.
