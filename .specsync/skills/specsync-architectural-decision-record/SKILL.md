# Architectural Decision Record

Follow the shared workflow in [../_shared/document-maintenance.md](../_shared/document-maintenance.md).

Review and own:

- `specs/03_Architecture-and-Design/Architectural-Decision-Record.md`

When the orchestrator agent asks for input:

- say whether the proposal affects this file
- list the file you reviewed
- if affected, describe the exact ADR delta to stage

Update this document when the request changes:

- architectural decisions and their status
- decision context or alternatives considered
- consequences and follow-up work
- references between decisions and architecture or design artifacts

Do not use this file as a generic changelog. Record only durable architectural decisions.
